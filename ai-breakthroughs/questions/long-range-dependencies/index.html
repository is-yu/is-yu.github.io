Can neural networks learn long-range temporal dependencies in sequences?

Context

Standard recurrent neural networks suffer from vanishing and exploding gradients, making it impossible to learn dependencies across many time steps. This was a fundamental barrier for sequence modeling.

Breakthroughs

Long Short-Term Memory
paradigm-shift 1997 · Sepp Hochreiter et al. · Neural Computation
LSTM solved the vanishing gradient problem in RNNs with gated memory cells, enabling learning of long-range dependencies in sequences.
