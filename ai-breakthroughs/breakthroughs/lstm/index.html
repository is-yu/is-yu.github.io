Long Short-Term Memory

Sepp Hochreiter, Juergen Schmidhuber

Summary

LSTM solved the vanishing gradient problem in RNNs with gated memory cells, enabling learning of long-range dependencies in sequences.

Key Findings

Finding 1
The LSTM architecture uses gated memory cells to selectively remember and forget information, solving the vanishing gradient problem for recurrent neural networks.
Enabled learning of dependencies spanning hundreds of time steps, making sequence modeling practical for real-world tasks.

Context

Hochreiter's 1991 diploma thesis identified the vanishing gradient problem. LSTM was designed specifically to address it, becoming the dominant architecture for sequence tasks for nearly two decades.
