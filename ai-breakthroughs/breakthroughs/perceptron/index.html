paradigm-shift 1958 Psychological Review

The Perceptron: A Probabilistic Model for Information Storage and Organization in the Brain

Frank Rosenblatt

Summary

The perceptron proved machines could learn pattern classification from labeled examples, launching the field of neural network research.

Key Findings

Finding 1
A single-layer neural network can learn to classify linearly separable patterns from labeled examples using a simple weight update rule.
First demonstration that a machine could learn from data, establishing the foundational concept of trainable artificial neural networks.
Finding 2
The perceptron convergence theorem guarantees that the algorithm will find a solution if one exists in finite steps.
Provided the first mathematical guarantee for a learning algorithm.

Context

In the late 1950s, AI research was dominated by symbolic approaches. Rosenblatt's perceptron offered a radically different paradigm: learning from examples rather than explicit programming.
