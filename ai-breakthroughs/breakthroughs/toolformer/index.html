Toolformer: Language Models Can Teach Themselves to Use Tools

Timo Schick, Jane Dwivedi-Yu, Roberto Dessi, Roberta Raileanu, Maria Lomeli, Luke Zettlemoyer, Nicola Cancedda, Thomas Scialom

Summary

Toolformer proved LLMs can self-teach tool use, enabling them to overcome inherent limitations by calling external APIs when needed.

Key Findings

Finding 1
Language models can learn to use external tools (calculator, search engine, translation API) in a self-supervised way, deciding when and how to call tools by generating special API call tokens.
Demonstrated that LLMs can autonomously decide when to augment themselves with tools, paving the way for agentic AI systems.

Context

Toolformer showed self-supervised tool learning was possible. Combined with function calling in GPT-4 and Claude, this enabled the explosion of AI agent frameworks in 2023-2024.
